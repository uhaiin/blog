## 关于我

你好，我是 uhaiin，一名热爱技术的 Java 研发工程师。

我一直深耕于后端开发领域，专注于 Java 技术栈及相关生态系统的实践与研究，包括 Spring Boot、Spring Cloud、MyBatis、JVM 原理、多线程、分布式架构、微服务治理等。在日常工作中，我参与过多个企业级系统的开发与维护，涉及统一客户资源平台、数据治理、配置管理、码表设计等多个模块，积累了丰富的实战经验。

这个博客是我记录学习历程、总结项目经验、分享技术思考的一片小天地。无论是开发中遇到的疑难杂症，还是对某个技术点的深入理解，亦或是工作中的日常感悟，我都会在这里一一写下。写作不仅帮助我梳理知识，也希望能在你遇到类似问题时，提供一点参考和启发。

技术之路永无止境，我始终保持好奇，持续学习，乐于交流。如果你也热爱编程，欢迎关注我的博客，或通过下方方式与我联系，一起探讨技术，共同成长！
