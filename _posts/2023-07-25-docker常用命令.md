> docker常用命令

### 1、查看正在运行的容器
```bash
docker ps
```
### 2、查看所有的容器
```bash
docker ps -a
```
### 3、停止容器
```bash
docker stop 123
```
### 4、删除容器
```bash
docker rm 123
```
### 5、查看镜像
```bash
docker images
```

### 6、删除镜像
```bash
docker rmi 123
```
